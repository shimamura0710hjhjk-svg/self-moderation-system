# self-moderation-system
コンテンツモデレーターのモデレーターの画面に少し似せたものを作りました。要するにBANか否かを決める体験ができるプログラムです。
動画で使用したプログラムの中身です。プライバシー保護と軽量化のためアセットの一部を除いてます。
STreamitを使用しています。
中身を入れ替える際は、ファイル名を変えた後、contents_id.jsonの中身のファイルパスを書き換えれば問題なく使えると思います。
このプログラムはGoogleGeminiを利用して作成しました。
【使用方法】
STreamitをインストールすれば必要なものがそろいます。
app.py　を実行すれば起動してローカルホストが立ち上がるはずです。そのページに飛べば使えるはずです。
もし、できなかったら…お手数をおかけしますがスクリーンショットを撮ってGeminiに聞いてください。エラー内容を読んで対処法を教えてくれます。
こんな投げやりな説明で申し訳ありません。初心者がGeminiに聞きながら作ったため私もよくわかってない状態です。
